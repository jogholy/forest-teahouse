# 🎨 森林茶屋视觉风格设计文档
**「旅行青蛙」式2D手绘治愈美学**

---

## 一、核心美术特征（对标分析）

| 特征维度 | 《旅行青蛙》风格 | 森林茶屋适配方案 |
|---------|----------------|----------------|
| 绘画手法 | 儿童简笔画+手绘质感 | 保留手绘笔触，增加水彩晕染效果 |
| 色彩基调 | 低饱和度、低对比度、柔和 | 森林绿+暖木色+雾感白，营造"晨间森林"氛围 |
| 线条风格 | 略带抖动的手绘线，不完美但亲切 | 粗铅笔线稿感，边缘微毛糙，拒绝矢量平滑 |
| 造型语言 | 高冷萌、二头身、眼神不看镜头 | 动物客人各有"性格萌点"，茶艺师（玩家化身）低调含蓄 |
| 场景构建 | 箱庭空间、自然材质家具 | 树洞茶屋+森林庭院，原生态木质结构 |

---

## 二、色彩系统（Color Palette）

### 🌿 主色调：「森林晨雾」

- **背景色**：`#F5F1E8`（米白宣纸感）
- **主色**：`#8B9A7C`（苔藓绿，占40%）
- **辅色**：`#D4C4B0`（原木色，占30%）
- **点缀色**：`#E8D4C4`（暖杏色，占15%）
- **文字色**：`#5C5C5C`（炭灰，非纯黑）

### 🎨 场景配色规则

- **室内茶屋**：原木色（70%）+ 深绿（20%）+ 陶土色（10%）
- **森林庭院**：草绿（50%）+ 晨雾白（30%）+ 花粉色（20%，随季节变）
- **UI界面**：半透明米白底（opacity 0.9）+ 深灰字 + 手绘图标

> **关键原则**：所有颜色HSL饱和度控制在20-40%，明度60-80%，营造"隔着毛玻璃看森林"的柔和感

## 三、角色设计规范

### 🐸 动物客人造型标准（「高冷萌」原则）

**基础比例：**
- 二头身（Q版），身体圆润无棱角
- 眼睛小且眼神"不直视玩家"（侧视或低垂），营造"独立生命感"
- 无复杂表情，情绪通过姿态和周围小物件暗示

**具体角色设计：**

| 角色 | 造型特征 | 性格暗示 | 专属配色 |
|-----|---------|---------|---------|
| 兔子记者棉棉 | 长耳微垂，背着帆布包，手持铅笔 | 好奇但害羞，耳朵朝向玩家但眼神飘向窗外 | 米白+浅棕 |
| 猫头鹰学者咕咕 | 圆眼镜（反光），翅膀抱书，站姿端正 | 知性疏离，眼镜反光遮住眼神 | 深褐+焦糖 |
| 狐狸旅行家阿狸 | 围巾遮半脸，背包挂满纪念品，尾巴蓬松 | 浪漫忧郁，侧影为主，尾巴表达情绪 | 橙红+暗灰 |
| 熊厨师憨憨 | 圆滚滚，围裙松垮，手持木勺 | 慵懒温和，半眯眼，嘴角微扬 | 棕黄+奶白 |

**手绘质感要求：**
- 填充色使用水彩笔刷，保留纸纹肌理
- 边缘线粗细不均，有"手抖"感
- 阴影用同色系深一号平涂，无渐变，保留块面感

## 四、场景设计：箱庭式空间

### 🏠 核心场景布局（仅3个主要场景，保持简约）

**场景1：茶屋室内（主界面）**

视角：侧视45°角，能看到屋内和窗外

结构：
```
├─ 左侧：原木茶台（操作区，泡茶动画在此）
├─ 中央：矮木桌+蒲团（客人落座区）
├─ 右侧：开放式木架（展示茶叶/茶具）
└─ 背景：圆窗透出森林景色（随现实时间变化光影）
```

**场景2：森林庭院（采集界面）**

视角：平视微俯，横向卷轴可滑动

元素：
```
├─ 左侧：茶屋入口（藤蔓门帘）
├─ 中央：小池塘（倒影有动态波纹）
├─ 采集点：发光植物（点击后有三叶草飞起特效）
└─ 右侧：小径通向深处（暗示更多区域待解锁）
```

**场景3：试茶台（研发界面）**

视角：俯视桌面

元素：
```
├─ 木质操作台（占画面70%）
├─ 食材格子（手绘图标，拖拽组合）
├─ 陶制茶具（试喝动画）
└─ 背景虚化，聚焦操作区
```

### 🌲 场景细节规范

- **材质表现**：所有木制品保留年轮纹理和结疤，不追求完美抛光
- **光影处理**：柔和顶光+侧窗光，阴影边缘模糊（水彩晕染效果）
- **动态元素**：
  - 室内：烛光摇曳（粒子效果）、窗帘微动
  - 室外：树叶飘落、雾气流动、偶尔有小鸟停留窗台

## 五、UI/UX设计：极简边缘化

### 📱 界面布局原则（参考《旅行青蛙》）

**核心规则：**
- UI占比<15%，最大化展示场景画面
- 所有按钮放在屏幕边缘，中央留白给角色互动
- 图标使用手绘简笔画，拒绝扁平化矢量感

**主界面布局：**
```
┌─────────────────────────────┐
│ [菜单]               [邮件]│ ← 顶部两侧，小图标
│                             │
│                             │
│     [主场景展示区]          │ ← 占80%画面，点击角色互动
│   （茶屋室内/庭院）         │
│                             │
│                             │
│                             │
│ [茶谱] [采集] [装饰] [好友]│ ← 底部，4个主按钮，手绘图标
└─────────────────────────────┘
```

### 🎨 UI元素设计规范

| 元素 | 设计风格 | 具体描述 |
|-----|---------|---------|
| 按钮 | 手绘边框+纸质感 | 圆角不规则，像剪贴纸片，按下时轻微变形 |
| 图标 | 简笔画风格 | 茶叶=几片叶子，茶谱=翻开的小本子，好友=两个牵手简笔人 |
| 文字 | 手写体/圆体 | 标题用"手写字体"（如方正清刻本悦宋），正文用圆体（如思源黑体圆） |
| 弹窗 | 信纸/便签形式 | 带胶带粘贴效果，边缘微卷，出现时有"纸张展开"动画 |
| 加载页 | 单幅插画+小动画 | 如"茶艺师正在采茶"的循环小动画，无进度条 |

## 六、动画与特效：克制而生动

### ✨ 动画原则

- **少即是多**：角色动画以微动为主（呼吸感、眨眼、耳朵微动），避免夸张表演
- **手绘感动画**：关键帧保留线条抖动，不追求平滑补间
- **粒子特效仅3种**：光点（采集时）、蒸汽（泡茶时）、花瓣（季节事件时）

### 🎬 关键动画设计

**泡茶流程动画（核心仪式感）：**

1. **取茶**：手伸向茶罐（特写手部，动作缓慢）
2. **温杯**：热水注入，蒸汽缓缓上升（半透明水彩效果）
3. **冲泡**：茶叶在水中舒展（2-3帧手绘变形）
4. **出汤**：茶汤倒入杯中，颜色渐变（从浅到深）
5. **呈上**：茶杯推至客人面前，客人手部入镜接过

**客人反应动画：**

- **满意**：耳朵/尾巴微动，头顶出现手绘小花（无表情变化，保持高冷）
- **惊喜**：身体微后仰，头顶出现"！"（手绘风格，非符号）
- **喝茶**：双手捧杯，蒸汽遮住半脸，只露眼睛（温馨感）

## 七、插画资产清单

### 🖼️ 必需美术资产

**场景背景（3张）：**
- 茶屋室内（四季变体：春樱/夏绿/秋枫/冬雪）
- 森林庭院（昼夜变体）
- 试茶台（固定）

**角色立绘（8-10个客人）：**
- 每个角色3个姿态：待机、喝茶、离开
- 每个角色1张"故事回忆"插画（解锁后展示，类似旅行青蛙的明信片）

**道具图标（30-40个）：**
- 食材类：茶叶（5种）、配料（8种）、水（3种）
- 茶具类：基础套装（3套）、限定套装（季节活动）
- 装饰类：家具（15件）、小摆件（10件）

**UI元素（1套）：**
- 按钮模板（5种状态：常态、悬停、按下、禁用、选中）
- 图标集（20个功能图标）
- 弹窗/面板背景（5种尺寸）

**特效动画（10个左右）：**
- 采集特效、泡茶特效、升级特效、季节特效等

## 八、技术实现建议

### 🛠️ 美术制作流程

**推荐工具链：**
- **原画/插画**：Photoshop + 水彩笔刷（Kyle's Watercolor系列）或 Procreate
- **动画**：Spine（2D骨骼动画，保留手绘质感）或逐帧动画（Toon Boom）
- **UI**：Figma（绘制后导出PSD手绘处理）或直接PS手绘

**关键Shader效果：**
- **纸纹叠加**：全屏叠加轻微纸张纹理（opacity 10-15%）
- **水彩边缘**：角色边缘轻微"晕染"效果（可用UV扰动实现）
- **柔光处理**：整体画面添加柔光层，降低对比度

**性能优化：**
- 静态场景使用大图拼接，动态元素单独分层
- 角色动画使用骨骼动画+蒙皮，减少逐帧成本
- 限制同屏粒子数量（<50个），保持"禅意"氛围

## 九、风格对比：与《旅行青蛙》的差异化

虽然对标《旅行青蛙》，但森林茶屋需要强化经营互动感：

| 维度 | 旅行青蛙 | 森林茶屋 |
|-----|---------|---------|
| 核心体验 | 等待与收集 | 创造与分享 |
| 角色关系 | 单向观察（蛙儿子不理你） | 双向互动（客人会回应你的茶） |
| 画面动态 | 极静（几乎无操作反馈） | 轻动（泡茶有流畅动画） |
| 色彩情绪 | 略带 loneliness（空巢感） | 温暖治愈（被需要的满足感） |
| 社交属性 | 纯单机 | 弱社交（异步茶会） |

---

**这个美术方案能帮你打造出既有《旅行青蛙》的治愈感，又有独特茶文化氛围的视觉体验。**
